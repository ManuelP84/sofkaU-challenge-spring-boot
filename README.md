### General Info
***
This is a Spring Boot back-end ToDo Application. In this API application you can perform the following operations:

* Add a category
* Add a new task
* Delete category and task
* Edit tasks

## Technologies
* Maven Project
* Java 11
* Spring Boot 2.6.7

## Dependencies
* starter-data-jpa
* starter-web
* mysql
* lombok
* validation
* test
